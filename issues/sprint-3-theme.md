## Overview

Create the Confessio design system with a dark warm palette inspired by candlelight
and parchment. Replaces the default Material3 theme generated by Android Studio.

## Changes required

### 1. `app/src/main/java/com/confessio/app/ui/theme/Color.kt`

```kotlin
val ConfessioBg          = Color(0xFF1A1208)
val ConfessioSurface     = Color(0xFF2C1F0E)
val ConfessioSurfaceAlt  = Color(0xFF3D2B14)
val ConfessioGold        = Color(0xFFD4A843)
val ConfessioGoldMuted   = Color(0xFF8B6914)
val ConfessioInk         = Color(0xFFF5ECD7)
val ConfessioMuted       = Color(0xFF8B7355)
val ConfessioError       = Color(0xFFB94040)
val ConfessioSuccess     = Color(0xFF4A7C4E)
```

### 2. `app/src/main/java/com/confessio/app/ui/theme/Theme.kt`

Define a `ConfessioTheme` composable using `MaterialTheme` with a dark color scheme:
- `background` → `ConfessioBg`
- `surface` → `ConfessioSurface`
- `primary` → `ConfessioGold`
- `onPrimary` → `ConfessioBg`
- `onBackground` → `ConfessioInk`
- `onSurface` → `ConfessioInk`
- `error` → `ConfessioError`

Wrap with `darkColorScheme(...)`. The composable signature:
```kotlin
@Composable
fun ConfessioTheme(content: @Composable () -> Unit)
```

### 3. `app/src/main/java/com/confessio/app/ui/theme/Type.kt`

Define a `Typography` using `MaterialTheme` defaults but with `ConfessioInk` color
for all text styles. Use system serif font for `displayLarge` and `headlineLarge`
(`fontFamily = FontFamily.Serif`). All other styles use default sans-serif.

## Acceptance criteria

- [ ] `Color.kt` defines all 9 color constants
- [ ] `ConfessioTheme` uses `darkColorScheme` with the correct color mappings
- [ ] `displayLarge` and `headlineLarge` use `FontFamily.Serif`
- [ ] `MainActivity.kt` wraps content in `ConfessioTheme`
